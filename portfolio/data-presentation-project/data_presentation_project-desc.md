### Data Presentation Project

__GOALS__ 
  -  Using datasets from OWID (Our World In Data) on the retirement ages of men and women across the world, plot the retirement ages of women in Japan, South Korea and the OECD average (treated as the global average) across a unified time period.

__NOTES__ 
  - All data cleaning and handling for this project was completed within Python. 
  - The `mock_csv.csv` and `plot_testing.py` files are test files for the code within this project, and are not expected to be functional.
